To run the following xlxs to csv you will need the following.

Please create the following directories::
Source directory path (This is where the source xlxs files live)
Destination directory path (This is where the csv files will be saved)

To run::
1.  Download the ps1 file.
2.  Set up your source/destination directories.
3.  Right click the script and run with powershell.
4.  Enter source path to files and press enter.
5.  Enter destination path for csv files and press enter.
6.  Wait for this process to complete and you should be prompted when to press enter once this has finished.
